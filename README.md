## Team Members
- Alan Hu
- Connor Hull

## Project Description
A web application for planning NYC day trips with friends, family, or partners. Users can explore activities, add them into a planner, save plans, review locations, and interact with friends.

## Tech Stack
- Node.js
- Express
- MongoDB
- Handlebars
- HTML/CSS/JavaScript

## Open Datasets
- Restaurants
- Museums
- Parks
- Cafes

## How to Run
1. npm install
2. npm start
